  详细对比

  🎯 Pencil MCP

  - 定位：专业的设计文件编辑器
  - 适用场景：需要精细控制设计节点、构建设计系统、管理组件库
  - 优势：
    - 精确控制每个设计元素
    - 支持变量和主题管理
    - 可通过代码批量操作
    - 适合构建可复用的设计系统

  🎯 Google Stitch

  - 定位：AI 驱动的快速 UI 生成工具
  - 适用场景：快速原型、设计灵感、开发者生成 UI
  - 优势：
    - 自然语言直接生成 UI
    - 支持图片转 UI（实验模式）
    - 直接输出生产级代码
    - 与 Figma 无缝集成
