## 内容长度

### 推荐字数

| 内容类型 | 字数范围 | 最佳字数 | 原因 |
|---------|---------|---------|------|
| 观点类 | 200-400字 | 300字 | 快速表达观点，引发讨论 |
| 教程类 | 300-600字 | 400-500字 | 步骤清晰，不宜过长 |
| 故事类 | 400-800字 | 500-600字 | 情节完整，保持悬念 |
| 列举类 | 300-500字 | 350-400字 | 要点清晰，易于扫描 |

**核心原则：**
- 能用3段说清楚，不写4段
- 能用短句，不用长句
- 删除所有废话和过渡词

## 内容结构

### 黄金3段式

```
第1段（开头）：2-3句话，必须吸睛
第2段（正文）：3-5句话，展开论证/步骤/故事
第3段（结尾）：1-2句话，强化记忆/行动号召
```

### 段落节奏

**短-长-短节奏：**
- 开头段：短句开场（1-2句）
- 正文段：长句展开（3-5句）
- 结尾段：短句收尾（1-2句）

**示例：**
```
【短】流量更重要，还是产品更重要？我坚定站在流量这边。

【长】为什么？因为我见过太多好产品死在没人知道上。一个老师做了10年课程，内容扎实，但推特只有200粉丝，月收入不到5k。一个律师写了3年公众号，专业度极高，但阅读量常年两位数。一个初创公司产品打磨了2年，技术领先，结果融不到钱倒闭了。

【短】流量为王！流量为王！流量为王！
```

## 视觉呈现

### 分段原则

**必须分段的情况：**
- 观点转换时
- 案例切换时
- 步骤分解时
- 强调重点时

**不要分段的情况：**
- 同一个观点的论证
- 同一个案例的展开
- 同一个步骤的说明

### 空行使用

**标准格式：**
```
段落1

段落2

段落3
```

**错误格式：**
```
段落1
段落2
段落3
```

## 标签使用

### 标签数量

- **推荐：** 0-2个标签
- **最多：** 3个标签
- **避免：** 超过3个（显得spam）

### 标签选择

**优先级：**
1. **热门话题标签**（如#AI #创业）
2. **行业标签**（如#自媒体 #内容创作）
3. **自创标签**（用于系列内容追踪）

**示例：**
```
好内容 #AI #创业
好内容 #自媒体
好内容（无标签也可以）
```

## 内容质量检查清单

### 发布前必查（5项）

- [ ] **开头是否吸睛**：前2-3句能否让人想继续看？
- [ ] **结构是否清晰**：分段合理？过渡自然？
- [ ] **有无真人感**：是否像AI生成的模板文？
- [ ] **错别字检查**：有无错字、标点错误？
- [ ] **数据是否真实**：提到的数字、案例是否准确？

---

## 禁止事项

### ❌ 绝对不要

1. **堆砌关键词**：不要为了SEO硬塞关键词
2. **标题党欺骗**：标题承诺的内容必须兑现
3. **抄袭他人内容**：可以借鉴思路，不能直接抄袭
4. **过度自嗨**：少用"牛逼""绝了"等极端词汇
5. **AI痕迹明显**：删除"在当今时代""综上所述"等AI惯用词

## 优化技巧

### 技巧1：删除废话

**删除前：**
> 那么，接下来我就来和大家分享一下我的看法。

**删除后：**
> （直接开始正文）

---

### 技巧2：短句开头

**优化前：**
> 我觉得在当今这个时代，流量真的是非常非常重要的一个因素。

**优化后：**
> 流量为王。

---

### 技巧3：用具体代替抽象

**优化前：**
> 很多人都有这个问题。

**优化后：**
> 我见过3个老师、2个律师、5个初创公司都有这个问题。

---

### 技巧4：用数字增加可信度

**优化前：**
> 效果很好。

**优化后：**
> 曝光从5w涨到80w。

## 真人感检查

### AI生成内容的特征（需避免）

**常见AI痕迹：**
- "在当今时代"
- "综上所述"
- "不仅...而且..."
- "首先...其次...最后..."
- "值得一提的是"
- "需要注意的是"

**改为真人语气：**
- 用短句
- 用口语
- 用反问
- 用个人经历
- 用具体案例
