# AI协作最佳实践

如何与Claude进行高效协作，最大化AI的价值，形成人机协作的最佳模式。

---

## 核心原则

**正确分工 = 人类（想法+拍板+出镜）+ AI（梳理+生成+管理）**

### 人类擅长
- ✅ 产生创意和灵感
- ✅ 做最终决策
- ✅ 提供真人感和情感
- ✅ 判断价值和质量

### AI擅长
- ✅ 梳理逻辑和结构
- ✅ 生成文稿和素材
- ✅ 文件管理和检索
- ✅ 数据分析和总结

## 第一层：对话技巧

### 1. 明确任务目标

❌ **不好的提示：** "帮我写一篇关于AI的文章"

✅ **好的提示：**
```
我要写一篇推特内容，主题是"流量大于产品"。

【目标】
说服初创公司创始人重视流量而非只关注产品。

【受众】
AI工具开发者、产品经理、初创公司创始人。

【要求】
- 长度：3-5个段落
- 结构：问题-观点-论证-强调
- 举例：老师、律师、初创公司三个案例
- 结尾：重复3次"流量为王"

请创作内容。
```

### 2. 提供上下文

❌ **不好的提示：** "优化这段文字"

✅ **好的提示：**
```
这是一篇推特内容的开头段落，目标是吸引注意力。

【原文】
今天我了解到了一些信息差变现的案例。

【问题】
开头不够吸引人，没有钩子。

【要求】
参考"反常识式"开头模板，改写成吸引人的开头。

【参考模板】
"你听我讲完就知道[某个现象]有多么[出乎意料的形容词]"
```

### 3. 渐进式提问

不要一次性提所有问题，分步骤对话：

**第1轮：** 确定选题方向
```
我有以下3个选题想法：
1. XX
2. XX
3. XX

请帮我分析哪个最有价值，为什么？
```

**第2轮：** 明确切入角度
```
我决定做"XX"这个选题。请帮我找出3个可能的切入角度。
```

**第3轮：** 生成内容大纲
```
我选择"XX"这个角度。请帮我生成内容大纲（开头-正文-结尾）。
```

**第4轮：** 创作完整文稿
```
基于这个大纲，请创作完整文稿。[附上可复用资产]
```

## 第二层：提示词模板

### 模板1：选题评估
```
我有一个选题想法：[选题内容]

请从以下维度评估：
1. 价值性（0-10分）：是否对读者有帮助？
2. 时效性（0-10分）：是否有时间窗口？
3. 差异化（0-10分）：是否有独特视角？
4. 素材完整性（0-10分）：是否有足够素材？

给出总分和建议。
```

### 模板2：内容创作
```
请根据以下信息创作一篇推特内容：

【选题】
[选题标题]

【目标受众】
[受众描述]

【内容结构】
- 开头：[使用XX开头模板]
- 正文：[展开方式]
- 结尾：[强化方式]

【可复用资产】
- 开头句式：[从爆款开头库选择]
- 核心概念：[从核心概念库选择]
- 金句：[从金句集锦选择]

【写作规范】
- 长度：3-5个段落，每段2-4句话
- 风格：真实、有观点、有价值
- 避免：AI味过重（"揭秘"、"深度解析"）

```

### 模板3：内容优化
```
这是一篇推特内容的初稿：

[原文]

请从以下维度优化：
1. 开头是否吸引注意力？如果不够，请改写。
2. 逻辑是否清晰？如果不清晰，请调整结构。
3. 有没有AI味过重的表达？如果有，请替换为真人化表达。
4. 金句是否足够强？如果不够，请增强表达。

```

## 第五层：避坑指南

### 常见错误

❌ **错误1：过度依赖AI**
- 问题：完全照搬AI生成的内容，没有人工审核
- 后果：内容失去真人感，互动率低
- 解决：AI生成后，必须人工审核和优化

❌ **错误2：指令不清晰**
- 问题："帮我写一篇文章"（目标、受众、要求都不明确）
- 后果：AI生成的内容不符合预期，需要多次迭代
- 解决：使用标准化提示词模板

❌ **错误3：不保存对话记录**
- 问题：与AI的对话没有记录，下次又要重新解释
- 后果：浪费时间，无法积累对话经验
- 解决：重要对话保存到"1-素材输入/对话记录/"

❌ **错误4：不迭代优化**
- 问题：AI第一次生成的内容不满意就放弃
- 后果：错过好选题，浪费灵感
- 解决：多次迭代，每次明确优化点

## 第七层：文科生入门AI心法（反常识）

基于"文科生和小白入门AI，十条极其反常识的心法"：

### 1. 用大于学
- 不要先学AI理论，直接开始用Claude协作
- 在实际创作中学习如何与AI对话

### 2. 万事问AI
- 不懂的概念，直接问Claude（而非百度）
- Claude会给出最符合你需求的解释

### 3. 不求助，自己思考
- 遇到问题，先自己尝试不同的提示词
- 多次迭代，找到最有效的对话方式

### 4. 认真对待AI
- 把Claude当成你的"第二大脑"
- 好好对话，提供清晰的上下文

### 5. 识别AI阿谀
- AI可能会过度肯定你的想法
- 使用特定提示词让AI给出客观分析：
```
请客观分析这个选题，指出可能的问题和风险，不要只说好话。
```

## 实战案例

### 案例1：从灵感到发布（流量大于产品）

**步骤1：碎片灵感**

```

```

AI：这是个很好的观点。请问：
1. 你有具体的案例吗？
2. 你认为流量和产品哪个更重要？

我：
1. 我看到很多官号只发产品，没有活人感
2. 我认为流量更重要
3. 目标受众是初创公司创始人和产品经理

AI：明白了。建议切入角度：
- 对比：流量 vs 产品
- 举例：老师、律师、初创公司的案例
- 结论：重复强调"流量为王"
```

```

**步骤4：人工审核和发布**
