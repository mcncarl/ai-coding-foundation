# 选题策划流程

基于"AI协作内容生产完整指南"的8步流程，标准化选题从灵感到发布的完整路径。

---

## 完整流程概览

```
步骤1：输入碎片内容 → 步骤2：对话澄清思路 → 步骤3：确定选题
↓
步骤4：生成文稿 → 步骤5：制作思维导图 → 步骤6：优化视频开头
↓
步骤7：生成标题封面 → 步骤8：录制视频
```

---

## 步骤1：输入碎片内容（素材积累）

### 来源渠道
- **日常灵感：** 随时记录想法到"1-素材输入/碎片灵感/"
- **推文收藏：** 看到爆款推文→保存到"推文收藏/"
- **阅读笔记：** 书籍、文章笔记→"阅读笔记/"
- **对话记录：** Claude对话精华→"对话记录/"

### 记录格式
```markdown
# [主题]

**灵感来源：** [X推文/书籍/对话]
**日期：** 2026-XX-XX
**核心想法：**
-
**可能的角度：**
-
**相关素材：**
-
```

### 质量标准
- ✅ 记录具体的想法，不是模糊的概念
- ✅ 标注灵感来源（方便追溯）
- ✅ 快速记录，不追求完美

---

## 步骤2：对话澄清思路（AI协助）

### 与AI对话的目标
1. 澄清选题的核心价值
2. 明确目标受众
3. 找到独特角度
4. 确定内容结构

### 对话模板
```
我有一个选题想法：[灵感内容]

请帮我：
1. 分析这个选题的核心价值是什么？
2. 目标受众是谁？
3. 有哪些可能的切入角度？
4. 哪个角度最有吸引力？
5. 需要补充哪些素材？
```

### 输出结果
- 明确的选题定位
- 2-3个可选角度
- 需要补充的素材清单

---

## 步骤3：确定选题（人类拍板）

### 评估维度

| 维度 | 评分标准 | 权重 |
|-----|---------|------|
| 价值性 | 是否对读者有帮助？ | 30% |
| 时效性 | 是否有时间窗口？ | 20% |
| 差异化 | 是否有独特视角？ | 20% |
| 素材完整性 | 是否有足够素材？ | 15% |
| 个人兴趣 | 是否愿意深入研究？ | 15% |

**总分>=70分 → 高优先级**
**总分50-70分 → 中优先级**
**总分<50分 → 低优先级**

### 创建选题文件
**命名规范：** `[优先级]-[类型]-[核心关键词].md`

示例：`高-观点-流量大于产品.md`

**文件模板：**
```markdown
# [选题标题]

**优先级：** 高/中/低
**类型：** 观点/教程/故事/列举/复盘
**目标受众：**
**核心价值：**
**独特角度：**

## 内容大纲
1. 开头：
2. 正文：
3. 结尾：

## 所需素材
- [ ]
- [ ]

## 可复用资产
- 爆款开头：
- 核心概念：
- 金句：
- 案例故事：

## 预期数据
- 目标曝光：
- 目标互动：
```

保存位置：`E:\obsidian\内容生产工厂\2-选题管理\待深化选题\[优先级]/`

## 步骤4：生成文稿（AI产出）

### 准备工作
1. 从"资产库"中寻找可复用内容：
   - 爆款开头库 → 选择开头句式
   - 核心概念库 → 复用已整理的概念
   - 素材数据库 → 引用案例和数据
   - 金句集锦 → 选择合适的金句

2. 整理输入给AI的信息：
   - 选题大纲
   - 可复用资产
   - 写作规范要求

### AI提示词模板
```
请根据以下信息创作一篇推特内容：

【选题】
[选题标题和核心价值]

【目标受众】
[受众描述]

【内容大纲】
1. 开头：[使用XX开头模板]
2. 正文：[展开方式]
3. 结尾：[强化方式]

【可复用资产】
- 开头句式：[从爆款开头库选择]
- 核心概念：[从核心概念库选择]
- 金句：[从金句集锦选择]
- 案例：[从案例故事库选择]

【写作要求】
- 长度：3-5个段落，每段2-4句话
- 结构：开头-正文-结尾
- 风格：真实、有观点、有价值
- 避免：AI味过重、标题党

请创作内容。
```

### 人工审核要点
- [ ] 开头是否吸引注意力？
- [ ] 逻辑是否清晰？
- [ ] 有没有错别字？
- [ ] 是否符合推特写作规范？
- [ ] 是否保留真人感？

## 步骤6：优化视频开头（视频内容）

### 视频开头黄金3秒
1. 问题式："你知道流量和产品哪个更重要吗？"
2. 反转式："今天我接了个1K的咨询，你听完就知道这个世界有多参差不齐。"
3. 数据式："中国还有十多亿人没用上AI。"

### 从文稿提取
- 使用文稿的开头句式
- 转化为口语化表达
- 增加视觉元素（文字动画、图片）

---

## 步骤7：生成标题封面（视频/长文）

### 标题公式
1. **数字式：** "2026年的4个小红书商机"
2. **问题式：** "流量更重要还是产品更重要？"
3. **反常识式：** "月入十万的PPT资料，你以为是红海？"
4. **护城河式：** "AI时代，你最大的护城河是什么？"

### 封面设计
- 关键词提取（1-3个核心词）
- 视觉元素（配图、数字）
- 人像元素（真人出镜截图）

---

## 步骤8：录制视频/发布内容（最终产出）

### 发布前检查清单
- [ ] 内容质量自检完成
- [ ] 视觉呈现优化完成
- [ ] 话题标签添加（1-3个）
- [ ] 发布时间选择合适（早/午/晚高峰）

### 发布后行动
- 30分钟内：回复所有评论
- 24小时内：主动@相关大V
- 48小时内：分析数据，记录经验

### 内容归档
**移动到：** `E:\obsidian\内容生产工厂\5-内容输出\推特帖子\[当月]/`

**同时更新：**
- 爆款内容库（如果互动量高）
- 数据统计（月度报告）
- 选题表现追踪

## 常见问题

### Q1：灵感太多，选不过来怎么办？
**A：** 用评估维度打分，优先做高分选题。低分选题放入"低优先级"，1个月后重新评估。

### Q2：创作时发现素材不足怎么办？
**A：** 暂停创作，先补充素材到"素材数据库"。素材完整性>=80%再继续创作。

### Q3：AI生成的内容不满意怎么办？
**A：** 检查输入是否完整（选题大纲+可复用资产+写作要求）。如果还不满意，多次迭代或人工重写。

### Q4：如何判断选题是否成功？
**A：** 发布后48小时，对比平均数据：
- 曝光>平均值的150% → 成功
- 曝光在平均值的80-150% → 正常
- 曝光<平均值的80% → 需要分析原因
