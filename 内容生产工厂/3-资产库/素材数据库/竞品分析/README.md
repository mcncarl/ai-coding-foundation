# 竞品分析库

从"爆款拆解"中提取的竞品成功要素汇总，按作者和主题分类，识别行业趋势和成功规律。

---

## 使用说明

### 1. 数据来源
- **一级来源：** `内容生产工厂/1-素材输入/爆款拆解/` 中的拆解文件
- **二级来源：** 自己的爆款内容分析（`4-数据统计/爆款内容分析.md`）

### 2. 分类方式

#### 按作者分类
为重点关注的作者创建独立文件：`[作者名]-竞品分析.md`

**适用场景：**
- 该作者有多个爆款值得深入研究
- 该作者的风格和定位与自己相似
- 该作者的粉丝画像与目标受众重合

#### 按主题分类
为高频主题创建独立文件：`[主题]-竞品分析.md`

**适用场景：**
- 某个主题有多个作者的爆款案例
- 想了解该主题的不同表达方式
- 计划在该主题深耕

#### 按时间分类（月度汇总）
每月末创建月度汇总：`YYYY-MM月度竞品分析.md`

**包含内容：**
- 本月拆解的所有爆款统计
- 共性规律总结
- 新趋势识别
- 下月策略建议

## 使用频率建议

### 每周（拆解后立即）
- 将拆解结果记录到对应的"按作者分类"或"按主题分类"文件
- 更新统计数据

### 每月末
- 生成月度汇总报告
- 分析趋势和规律
- 制定下月策略

### 每季度
- 复盘3个月的竞品分析
- 识别长期趋势
- 调整年度内容策略

---

## 注意事项

### ✅ DO（应该做）
- 只拆解真正的爆款（曝光>50w 或 互动>1000）
- 聚焦可复用要素，不要只是记录
- 立即思考如何应用到自己的选题
- 定期更新和汇总

### ❌ DON'T（不应该做）
- 不要简单模仿，要差异化
- 不要拆解低质量内容，浪费时间
- 不要只收集不应用
- 不要忽视自己的优势和特点
