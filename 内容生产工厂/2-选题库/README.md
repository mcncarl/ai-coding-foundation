# 选题库使用指南

**重要：** 这不是创作前的待办清单，而是爆款复盘分析库！

---

## 选题库是什么？

**选题 = 爆款复盘分析**

- ❌ **不是**：创作前的规划文档
- ✅ **是**：发布后对爆款内容的成功要素分析
- **作用**：当没有灵感时，从选题库找爆款案例，获取新灵感

---

## 使用场景

### 场景1：没有灵感时（主要用途）

```
没有灵感 → 打开选题库 → 按内容类型浏览 → 找爆款案例 → 获取新灵感
```

**操作步骤：**
1. 想写观点？→ 查看"观点思维类/"
2. 想分享工具？→ 查看"工具推荐类/"
3. 想写教程？→ 查看"教程指南类/"
4. 想讲商机？→ 查看"商机赚钱类/"
5. 查看爆款案例的成功要素
6. 获得新灵感 → 直接写

---

### 场景2：创作同类内容时

```
创作中 → 打开同类型选题 → 复用成功要素（开头/结构/金句）
```

---

### 场景3：优化内容时

```
内容写完 → 参考爆款结构 → 优化内容质量
```

---

## 文件夹分类

### 观点思维类/
**包含内容：** 表达观点、传递思维方式的内容

**代表案例：**
- 流量大于产品
- 不是所有的钱都要赚
- 先吃饱饭再谈道
- 贵就是便宜，便宜就是贵

**特点：**
- 争议性观点
- 反常识思维
- 价值观碰撞

---

### 商机赚钱类/
**包含内容：** 发现商机、分享赚钱方向的内容

**代表案例：**
- 2026的四个小红书商机
- B端老板的钱很好赚
- 小红书英语漫画
- 广告背后都是商机

**特点：**
- 具体商机
- 变现路径
- 案例驱动

---

### 工具推荐类/
**包含内容：** 工具评测、功能介绍、工具对比

**代表案例：**
- Obsidian+Claude Code
- 浏览器自动化
- deepseek
- Antigravity Agent Manager

**特点：**
- 工具介绍
- 功能评测
- 效率提升

---

### 教程指南类/
**包含内容：** 配置教程、安装指南、步骤详解

**代表案例：**

- Claude Code对话精华导出Obsidian配置

**特点：**
- 步骤详细
- 可操作性强
- 完整指南

---

### 方法技巧类/
**包含内容：** 提示词、学习方法、效率技巧

**代表案例：**
- ASCII图复习法学

**特点：**
- 技巧分享

---

### 生活故事类/
**包含内容：** 个人经历、案例分享、生活观察

**代表案例：**
- AI和自媒体的具体的"路"
- 酒店怪相
- 围城

**特点：**
- 生活化
- 故事性

---

### 技术文档类/
**包含内容：** Git规范、编程规范、技术标准

**代表案例：**

**特点：**

- 参考文档

## 选题文件格式

**文件命名：** `[文章标题]-选题分析.md`

**文件结构：** 见选题分析模板

---

## 什么内容需要创建选题分析？

**标准：**
- ✅ 数据好的内容（曝光>10w 或 互动>500）
- ✅ 自己觉得写得好的内容
- ✅ 可以提取成功要素的内容

**不需要：**
- ❌ 数据差的内容（除非有特殊启发）
- ❌ 试验性内容
- ❌ 临时性内容

---

## 创建选题分析的时机

**时机：** 发布后48小时，数据稳定

**操作步骤：**
1. 查看发布内容的数据（曝光、互动）
2. **数据好的内容**：
   - 复制"选题分析模板"
   - 填写快速信息（类型、数据、原文链接）
   - 分析成功要素（为什么成功？）
   - 提取可复用要素（开头/结构/金句）
   - 列出灵感来源
3. 保存到对应的内容类型文件夹
