# 核心概念说明 v3.0（精简版）

---

## 核心4概念

| 概念 | 定义 | 示例 | 时机 |
|-----|------|------|------|
| **灵感** | 模糊想法 | "我觉得流量很重要" | 随时 |
| **选题** | 爆款复盘分析（事后） | "流量大于产品"为什么成功？ | 发布后数据好时 |
| **标题** | 吸引人的名字 | "流量更重要，还是产品更重要？" | 文稿完成后 |
| **内容** | 完整推文文稿 | 300字完整推文 | 可直接发布 |

---

## 关键区别

**选题的定义（核心修正）：**
- 选题是发布后的爆款复盘分析（事后分析）
- 选题不是创作前的规划文档（事前规划）
- 灵感来了就直接写，不需要先做选题

**资产库 vs 内容库：**
- 资产库 = 可复用的积木块（开头句式、金句、结构模板）→ 不能直接发布
- 内容库 = 完整作品（推文文稿）→ 可以直接发布
- 判断标准：**能直接发布吗？能 → 内容库，不能 → 资产库**

---

## 工作流

**主流程（灵感驱动）：**
```
灵感 → 直接写 → 发布 → 数据追踪（48h后）→ 数据好 → 选题分析 → 提取要素到资产库
```

**没灵感时：**
```
打开选题库 → 按分类浏览爆款案例 → 获取灵感 → 直接写
```

---

## 选题的正确用法

| 场景 | 操作 |
|-----|------|
| 没有灵感 | 翻选题库 → 看爆款案例 → 获取灵感 |
| 创作同类内容 | 查看同类型选题 → 复用成功要素 |
| 优化内容 | 参考爆款结构 → 提升内容质量 |

**选题不用于：** 创作前规划、管理优先级、提前设计大纲

---

## 常见误区

| 误区 | 正确理解 |
|-----|---------|
| 必须先做选题才能写 | 灵感来了就直接写 |
| 选题 = 标题 | 选题是爆款配方分析，标题是吸引人的名字 |
| 选题是事前规划 | 选题是事后复盘，数据好的内容才做 |

---

**核心记忆：** "选题是爆款配方分析，不是创作前的规划。灵感来了就直接写！"
