## 核心概念（必读）

| 概念 | 定义 | 时机 |
|-----|------|------|
| **灵感** | 模糊想法 | 随时记录 |
| **选题** | 爆款复盘分析（事后） | 发布后数据好时 |
| **标题** | 吸引人的名字 | 文稿完成后 |
| **内容** | 完整推文文稿 | 可直接发布 |

**关键规则：**
- 灵感来了就直接写，不需要先做选题规划
- 选题是事后复盘（不是事前规划），用于没灵感时获取灵感
- 资产库 = 可复用积木块（不能直接发布）
- 内容库 = 完整作品（可以直接发布）

## 创作工作流

**主流程（灵感驱动）：**
```
灵感 → 直接写 → 发布 → 数据追踪（48h后）→ 数据好？→ 选题分析 → 提取要素到资产库
```

**没灵感时：**
```
打开选题库 → 按分类浏览爆款案例 → 获取灵感 → 直接写
```

## AI协作规则

### 禁止
- 删除任何文章（即使数据差）
- 修改已发布的文章（除非明确要求）
- 改变文件夹结构（除非明确要求）

### 可以
- 创建新文件
- 编辑草稿
- 生成分析报告

### 创作风格
- 前2-3句必须吸睛（问题式/反常识/信息差）
- 口语化、短句、接地气
- 数字具体化（不说"很多"，说"10倍"）
- 善用重复强调（核心观点重复3次）
