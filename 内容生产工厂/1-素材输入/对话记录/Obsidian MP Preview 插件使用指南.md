## 插件简介

MP Preview 是一个帮助你快速将 Obsidian 笔记转换为微信公众号格式的插件。它提供优雅的模板、可自定义的字体和样式、实时预览功能，以及直接复制到剪贴板的便捷编辑和发布体验。

**最大亮点**：无需配置图床，本地图片会被自动编码，粘贴到公众号后自动上传到微信素材库，真正实现本地化工作流。

## 核心功能

- **实时预览**：在 Obsidian 内直观查看公众号效果，所见即所得
- **丰富模板**：7 种精美样式模板可选
- **字体定制**：5 种字体选择，字号 12-30 号可调
- **本地图片自动处理**：图片通过特定方式编码，粘贴后自动上传，无需图床
- **一键复制**：复制到剪贴板后，文字、样式、图片全部保留
- **锁定预览**：避免刷新打断写作流程
- **完整 Markdown 支持**：代码块、表格、图片等元素完美转换

## 安装步骤

### 方法一：插件市场安装（推荐）

1. 打开 Obsidian，点击左下角的 **设置图标（⚙️）**
2. 在左侧菜单找到 **第三方插件**
3. 如果是第一次使用，需要先 **关闭安全模式**
4. 点击 **浏览社区插件**
5. 在搜索框输入 **"MP Preview"**
6. 找到插件后点击 **安装**，然后点击 **启用**

### 方法二：手动安装

1. 访问 GitHub 仓库：https://github.com/Yeban8090/mp-preview
2. 下载最新的 Release 版本
3. 解压到 `.obsidian/plugins/` 目录下
4. 重启 Obsidian，在设置中启用插件

## 使用方法

### 1. 打开预览窗口

- 在 Obsidian 中打开你要发布的 markdown 文档
- 按 `Ctrl + P`（Mac 用户按 `Cmd + P`）打开命令面板
- 输入 **"MP Preview"** 并选择命令
- 右侧会打开一个实时预览窗口

### 2. 调整样式

在预览窗口中可以自定义：
- **模板样式**：7 种精美模板任选
- **字体选择**：5 种字体可选
- **字号调整**：12-30 号范围
- **背景颜色**：根据个人喜好设置

### 3. 复制到公众号

1. 调整样式到满意的效果
2. 点击预览窗口中的 **"一键复制"** 按钮
3. 打开微信公众号后台编辑器
4. 直接 `Ctrl + V`（或 `Cmd + V`）粘贴
5. **所有格式、文字、图片都会自动保留**，无需再处理

### 4. 图片处理说明

- **本地图片**：插件会自动编码，粘贴到公众号后会自动上传到微信素材库
- **网络图片**：直接引用 URL 即可
- **无需图床**：不依赖第三方图床服务，避免图片失效风险

## 常见问题

### Q: 图片会保存在哪里？
A: 本地图片会被编码后嵌入到复制的内容中，粘贴到公众号后会自动上传到微信素材库。

### Q: 支持哪些 Markdown 语法？
A: 支持常见的 Markdown 语法，包括标题、列表、代码块、表格、图片、链接、引用等。

### Q: 可以自定义 CSS 样式吗？
A: 插件提供了 7 种内置模板，如需更深度定制，可以查看插件的 GitHub 仓库了解自定义方法。

### Q: 粘贴到公众号后还需要调整吗？
A: 通常不需要，格式和图片都会完美保留。如有微调需求，可在公众号编辑器中直接修改。
