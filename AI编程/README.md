# AI编程学习笔记

> 教你怎么用AI帮你写代码

---

## 这是什么？

这个文件夹里装的是**AI编程教程**。

简单说，就是教你怎么让AI帮你写程序。

就像请了一个程序员小助手，你告诉它要做什么，它就帮你写代码。

---

## 里面有什么？

| 文件 | 有什么用？ |
|------|-----------|
| 1.AI编程入门.md | 先告诉你AI是怎么"学会"写代码的 |
| 2.提示工程实战.md | 教你怎么跟AI说话，它才听得懂 |
| 3.AI编程应用.md | 教你怎么让AI帮你干活 |
| 4.AI与未来.md | 告诉你AI以后会变成什么样 |

---

## 怎么学？

就像看连环画一样，按顺序看就行：

```
第1步：先看看AI是怎么工作的
    ↓
第2步：学会怎么跟AI说话
    ↓
第3步：开始让AI帮你写代码
    ↓
第4步：看看未来会变成什么样
```

---

## 谁适合看？

- ✅  完全不会编程的小白
- ✅  想用AI帮忙写代码的人
- ✅  对AI感兴趣的好奇宝宝

---

## 怎么用？

1. 打开文件，从第一个开始看
2. 跟着例子操作试试
3. 有不懂的，复制给AI问问

---

## 举个例子

**以前你要自己写代码：**
```
你 → 苦思冥想 → 写代码 → 累个半死
```

**现在让AI帮你：**
```
你：帮我写一个计算器
AI：好的！噼里啪啦...（代码写好了）
你：真棒！
```

- 好奇心一个
