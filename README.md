## ⭐ v2.0 核心修正（2026-02-13）

**关于"选题"的正确理解：**
- ❌ **旧理解：** 选题是创作前的规划文档（事前做选题 → 再写文章）
- ✅ **新理解：** 选题是发布后的爆款复盘分析（写文章 → 发布 → 数据好 → 做选题分析）

**"选题"的真正作用：**
- **不是**：创作前必须做的规划（太僵化，打断创作流）
- **而是**：发布后对爆款内容的成功要素分析
- **目的**：当没有灵感时，从选题库找爆款案例，获取新灵感

**工作流变化：**
- ❌ 旧流程：做选题规划 → 写文章 → 发布
- ✅ 新流程：灵感来了 → 直接写 → 发布 → 数据追踪 → 爆款复盘（选题分析）

### 日常创作流程（灵感驱动）⭐

**工作流1：有灵感时（主流程）**
1. **直接开始写**：灵感来了就马上写，不要先做选题规划！
2. **快速复用资产**：从"3-资产库/"快速查阅可复用内容
   - 爆款开头库 → 参考开头句式
   - 核心概念库 → 复用已整理的概念
   - 金句集锦 → 引用金句
3. **AI协作**：使用AI生成初稿（参考[[AI协作最佳实践]]）
4. **人工审核**：检查真人感、逻辑、错别字
5. **发布归档**：发布后保存到"5-内容输出/推特帖子/当月/"
6. **数据追踪**：48小时后记录数据到"4-数据统计/"

**工作流2：没有灵感时（辅助流程）**
1. **打开选题库**：浏览"2-选题库/"按内容类型查找
   - 想写观点？→ 查看"观点思维类/"
   - 想分享工具？→ 查看"工具推荐类/"
   - 想写教程？→ 查看"教程指南类/"
   - 想讲商机？→ 查看"商机赚钱类/"
   - 想分享方法？→ 查看"方法技巧类/"
   - 想讲故事？→ 查看"生活故事类/"
2. **查看爆款案例**：找到数据好的内容，分析成功要素
3. **获得新灵感**：从爆款案例中获取灵感 → 回到工作流1直接写

**核心原则：灵感来了就写，不要被"选题规划"束缚！**

### 每周数据复盘与爆款拆解（周日）⭐

**1. 复盘本周发布内容（事后分析）**
- 查看本周发布内容的数据（曝光、互动）
- **数据好的内容**（曝光>10w 或 互动高）：
  - 创建"选题分析文件"（按内容类型保存到"2-选题库/"）
  - 分析成功要素（为什么成功？）
  - 提取可复用要素（开头/结构/金句）→ 存入"3-资产库/"
- 数据差的内容：分析原因，优化下次创作

**2. 拆解他人爆款**
- 选择3-5个他人爆款推文进行拆解（使用"爆款拆解模板"）
- 提取要素：将可复用要素存入"资产库"（开头、结构、金句）

**3. 整理碎片灵感**
- 从"1-素材输入/"中筛选有价值的灵感
- 记录在"碎片灵感/"中，等待灵感自然发酵

**关键区别：**
- ❌ 不是：事前做选题规划，安排下周写什么
- ✅ 而是：事后分析爆款，提取成功要素到资产库
- **目的：** 让资产库越来越丰富，下次创作越来越容易

### 每月复盘（月末）
1. 生成月度报告（使用"4-数据统计/月度报告模板.md"）
2. 分析爆款内容，提取要素到"3-资产库/"
3. 更新"方法论文档"
4. 设定下月目标

## 💡 常见问题

### Q1：如何开始第一篇内容创作？
**A：**
1. **有灵感时**：直接写！参考"推特写作规范.md"和"爆款开头库.md"
2. **没有灵感时**：打开"2-选题库/"浏览爆款案例，获取灵感后直接写
3. 使用AI协作（参考"AI协作最佳实践.md"）
4. 发布后记录数据到"4-数据统计/"
5. 数据好的内容（曝光>10w），做选题分析并提取要素到资产库

**关键：不要先做选题规划，灵感来了就直接写！**

### Q1.5：如何拆解他人爆款推文？
**A：**
1. 找到真正的爆款（曝光>50w 或 互动>1000）
2. 使用"爆款拆解模板"（5分钟完成）
3. 聚焦3个核心要素：开头、结构、金句
4. 可用AI辅助分析（将推文内容粘贴给Claude）
5. 立即提取可复用要素，存入"资产库"

### Q2：如何提升创作效率？
**A：**
1. 每次创作都从"资产库"中复用内容
2. 建立标准化提示词模板（参考"AI协作最佳实践.md"）
3. 不要追求完美，先发布再优化
4. 每月复盘，淘汰失效策略，强化有效策略

### Q3：如何判断内容是否成功？
**A：**
发布后48小时，对比平均数据：
- 曝光>平均值的150% → 成功
- 曝光在平均值的80-150% → 正常
- 曝光<平均值的80% → 需要分析原因

### Q4：资产库太空了怎么办？
**A：**
两种方法快速积累：

**方法1：从自己的内容中提取（慢但精准）**
- 前10篇内容：主要是积累，复用率低
- 10-30篇内容：开始复用，效率提升
- 30篇+：复利效应显现，效率大幅提升

**方法2：从他人爆款中提取（快速启动）**
- 每周拆解3-5个爆款推文
- 4周就能积累12-20个案例
- 边拆解边学习，边应用边优化
- 推荐：先拆解10个爆款，快速建立资产库基础
