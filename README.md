1. 阅读 [[AI协作内容生产-完整指南]]（设计文件/Excalidraw/）

### 日常创作流程
1. **选择选题**：打开"2-选题管理/待深化选题/高优先级/"
2. **复用资产**：从"3-资产库/"中寻找可复用内容
3. **AI协作**：使用AI生成初稿（参考[[AI协作最佳实践]]）
4. **人工审核**：检查真人感、逻辑、错别字
5. **发布归档**：发布后移动到"5-内容输出/推特帖子/当月/"
6. **数据追踪**：48小时后记录数据到"4-数据统计/"

### 每周选题策划（周日）
1. 从"1-素材输入/"中筛选有价值的灵感
2. 与AI对话澄清思路
3. 在"2-选题管理/待深化选题/"创建3-5个新选题
4. 标注优先级和所需素材

### 每月复盘（月末）
1. 生成月度报告（使用"4-数据统计/月度报告模板.md"）
2. 分析爆款内容，提取要素到"3-资产库/"
3. 更新"方法论文档"
4. 设定下月目标

## 💡 常见问题

### Q1：如何开始第一篇内容创作？
**A：**
1. 阅读"推特写作规范.md"
2. 打开"待深化选题/高优先级/"，选择一个选题
3. 参考"爆款开头库.md"选择开头句式
4. 使用AI协作（参考"AI协作最佳实践.md"）
5. 发布后记录数据

### Q2：如何提升创作效率？
**A：**
1. 每次创作都从"资产库"中复用内容
2. 建立标准化提示词模板（参考"AI协作最佳实践.md"）
3. 不要追求完美，先发布再优化
4. 每月复盘，淘汰失效策略，强化有效策略

### Q3：如何判断内容是否成功？
**A：**
发布后48小时，对比平均数据：
- 曝光>平均值的150% → 成功
- 曝光在平均值的80-150% → 正常
- 曝光<平均值的80% → 需要分析原因

### Q4：资产库太空了怎么办？
**A：**
前期资产库需要积累：
- 前10篇内容：主要是积累，复用率低
- 10-30篇内容：开始复用，效率提升
- 30篇+：复利效应显现，效率大幅提升
